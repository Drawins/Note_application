# ToDo Application

This is a simple ToDo application that allows users to manage their tasks.

## Features

- Add new tasks
- View existing tasks
- Delete tasks

## Installation

1. Clone the repository: https://github.com/Drawins/Note_application.git


## Usage
- Click on the "+" button to add a new task.
- Click on a task to navigate to the update and delete page.
- View the task on a list.

## Added Functionality

- Users can now add new todo items.
- Implemented backend logic to handle updating the task status in the database.
- Implemented frontend logic to reflect the changes in the UI.
- Added functionality to delete todo items if the user wishes to do so.
